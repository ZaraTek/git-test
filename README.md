# Heading 1 Remember commands for initializeing Git repository:

1. git init # first step to create repo
2. git branch -M main # rename branch to main to avoid confusion with master
3. git add README.md # always include read me file
4. git remote add origin "github_link" # connects local repo to the remote github repo names "origin"
5. git push -u origin main # tells future pushes to push to origin
